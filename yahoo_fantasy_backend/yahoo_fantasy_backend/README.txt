1. Replace YOUR_REFRESH_TOKEN in app.py with your refresh token from Yahoo OAuth.
2. Push this project to GitHub.
3. Deploy on Railway as a Python app.
4. Railway auto-detects Flask from requirements.txt.
5. Update ai-plugin.json with your Railway URL.
6. Install the plugin in ChatGPT.
7. Test endpoints:
   - /players
   - /teams
   - /matchups
